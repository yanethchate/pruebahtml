​ Poco después, otros pioneros como Gottlieb Daimler y Wilhelm Maybach presentaron sus modelos. El primer viaje

 Un motor es la parte sistemática de una máquina capaz de hacer funcionar el sistema,
transformando algún tipo de energía (eléctrica, de combustibles fósiles, etc.), en energía
mecánica capaz de realizar un trabajo. En los automóviles este efecto es una fuerza que produce el movimiento. 		El sistema frenos antibloqueo (ABS) es un sistema de frenado que evita
que las ruedas se bloqueen y patinen al frenar, con lo que el vehículo
no solamente decelera de manera óptima, sino que permanece estable y
direccionable durante la frenada (podemos girar mientras frenamos).
Fue diseñado para ayudar al conductor a mantener cierta capacidad de dirección
y evitar el arrastre durante el frenado. Con el sistema ABS se impide que ninguna de las 4 ruedas patine, lo que permite
dirigir el vehículo y seguir manteniendo el frenado (frenar y dirigir al mismo tiempo)
